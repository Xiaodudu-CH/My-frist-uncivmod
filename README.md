# 华夏帝国 - Unciv 模组

## 文明特色
**领袖：** 始皇帝
**特色能力：科技强军**
- 所有科研产出 **+100%**
- 所有远程单位 **射程+1，移动力+2，经验获取+50%**

**特色单位：**
1.  **秦锐士** (替代剑士)：可以攻击后移动，免疫近战减益的强力先锋。
2.  **诸葛连弩** (替代弩手)：受文明能力加持，拥有超远射程和机动性的远程核心。

**特色建筑：**
1.  **稷下学宫** (替代图书馆)：除了提供科研，还能额外增加食物，促进人口与科技同步增长。
2.  **天工坊** (替代工坊)：提供强大的产能加成，并与伐木场形成联动。

## 安装方法
1.  在 Unciv 游戏主界面，点击 **“模组”**。
2.  点击 **“下载模组”**。
3.  在搜索框中输入 **`华夏帝国`** 或 **`Xiaodudu-CH`** 进行搜索。
4.  找到模组后，点击右侧开关下载并启用。

## 文件结构
本模组完全开源，主要文件位于 `jsons/` 目录下：
- `Nations.json`: 文明定义与全局能力
- `Units.json`: 特色单位定义
- `Buildings.json`: 特色建筑定义
